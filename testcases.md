# Лабораторная работа №1 по дисциплине «ПА СМ»
**Преподаватель:** доц., к.т.н., доц. Жаринов О.О.

## Задание
Написать программу, формирующую видео, в котором по изображению хаотически перемещается окружность. Для пикселей, попавших внутрь окружности, устанавливается нулевая насыщенность цвета, для остальных цвет сохраняется.

## Использованные инструменты
* **Язык:** Python 3.14
* **Библиотеки:** OpenCV (cv2), NumPy
* **IDE:** Visual Studio Code
