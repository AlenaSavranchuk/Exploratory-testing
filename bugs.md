# Баг-репорт: Некорректная локализация кнопки личного кабинета

**URL сайта:** [https://ftb.mentorpiece.org/?lang=ru](https://ftb.mentorpiece.org/?lang=ru)

| ID | 1_ИТ |
| :--- | :--- |
| **Краткое описание** | Кнопка "User" в правом верхнем углу на главной странице после авторизации отображается на английском языке при выбранном русском интерфейсе. |
| **Окружение** | Windows 11 Home, Chrome 144.0.7559.133 (64-bit) |

### Подробное описание
Пользователь выбрал русский язык на главной странице и авторизовался. После перенаправления на главную страницу кнопка перехода в личный кабинет сохраняет англоязычное название "User".
- **Фактический результат:** Надпись на кнопке — "User".
- **Ожидаемый результат:** Надпись на кнопке — "Пользователь" (или аналогичный перевод на русский).

### Шаги по воспроизведению
1. Открыть сайт [https://ftb.mentorpiece.org/?lang=ru](https://ftb.mentorpiece.org/?lang=ru).
2. В форме входа ввести валидные логин и пароль.
3. Нажать кнопку **"Войти"**.
4. Обратить внимание на кнопку в правом верхнем углу главной страницы.
